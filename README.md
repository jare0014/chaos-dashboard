# The Chaos Dashboard

A Python simulation of chaotic systems, built to explore:
1. **The Mandelbrot Set** (Escape Time Algorithm)
2. **Polymer Generator** (Scaling Laws)
3. **Double Pendulum Dynamics** (Sensitive Dependence)

## Tech Stack
* Python 3.12+
* NumPy
* Matplotlib
* Streamlit
* Scipy
* Plotly
